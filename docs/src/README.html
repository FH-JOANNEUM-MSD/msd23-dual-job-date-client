Projekt

Dual Job Dating ist eine plattformübergreifende Mobile-App (Android & iOS) für das Job-Dating-Event der FH Joanneum. Studierende bewerten Partnerunternehmen per Like / Dislike / Neutral; basierend auf den Voting-Ergebnissen weist das Backend Termin-Slots zu, die die Studierenden in der App einsehen. Die App ist in Kotlin Multiplatform mit Compose Multiplatform umgesetzt; das Backend (nicht Teil dieser Abgabe) läuft als Go-Service gegen Supabase.

Externe Ressourcen

Quellcode (Git-Repository)

https://github.com/FH-JOANNEUM-MSD/msd23-dual-job-date-client

Der gesamte App-Quellcode liegt in diesem Repository. Branches: main (Entwicklung), android-testing (Build → Internal Testing), android-production (Build → Closed Testing Alpha). Dokumentation befindet sich unter docs/; der App-Quellcode unter composeApp/src/.

Ticketverwaltung (YouTrack)

https://sicherheitspasskmp.youtrack.cloud/projects/JOB

Projektschlüssel JOB. Enthält alle User Stories, Tasks und Bugs der drei Sprints (100 erledigte Tickets gesamt). Workflow: Waiting → To do → In Progress → Done. Tickets sind mit den Pull Requests im Repository verlinkt.

App-Verteilung

Android - Google Play Closed Testing (Alpha):

https://groups.google.com/g/dualjobdating

Studierende treten der Google-Gruppe bei und erhalten anschließend Zugriff auf die App im Play Store. Build wird automatisch bei jedem Push auf android-production aus der GitHub-Actions-Pipeline veröffentlicht.

iOS - TestFlight (öffentlicher Einladungslink):

https://testflight.apple.com/join/eJBwfHqd

Studierende installieren TestFlight aus dem App Store, öffnen den Einladungslink und installieren die App. TestFlight-Builds laufen nach 90 Tagen ab.
